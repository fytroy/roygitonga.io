Building My Personal Automation Dashboard with Streamlit and Python

Published on June 13, 2025 | Category: Python, Automation, AI | By Rodney Gitonga

Welcome to my journey of creating a personal automation dashboard using Streamlit and Python! In today's fast-paced world, automating repetitive tasks and centralizing information can significantly boost productivity. This project was born out of a desire to have a single, intuitive interface for various daily needs, from tracking cryptocurrency prices to getting real-time weather updates and even leveraging AI for content summarization.

The Inspiration and Goal

I often found myself jumping between multiple tabs and applications for simple tasks: checking crypto, weather, system health, or quickly sending an email. The idea was to consolidate these functionalities into one sleek, interactive dashboard. Streamlit, with its ability to quickly turn Python scripts into beautiful web apps, was the perfect tool for this.

Key Features and Technologies Used:

  • Cryptocurrency Tracking: Integrated with CoinDesk and CoinGecko APIs to display real-time prices and trends for popular cryptocurrencies.
  • Weather Updates: Utilized the OpenWeatherMap API to fetch current weather conditions for a specified location.
  • Website Monitoring: A simple uptime checker to ensure my personal projects and other important sites are live.
  • System Resource Display: Used `psutil` to monitor CPU, memory, and disk usage directly from the dashboard.
  • Google Drive Backup: A quick utility to backup specified local folders to Google Drive, leveraging the Google Drive API.
  • Email Sender: A simple form to send quick emails without opening a mail client.
  • AI-Powered News & Document Summarization: This is where the Google Gemini API comes in. Users can input text or URLs, and Gemini summarizes them, saving time on reading lengthy articles.

Challenges and Solutions

One of the initial challenges was managing API keys securely and handling different API response formats. I implemented environment variables for sensitive keys and created modular functions for each API integration to keep the code clean and manageable.

Another interesting aspect was ensuring the Streamlit app was responsive and visually appealing. Streamlit's intuitive layout options helped, and I used custom CSS (linked from `../css/style.css`) to fine-tune the aesthetics to match my portfolio's theme.

Future Enhancements

I'm always looking to expand this dashboard. Future plans include integrating task management, calendar synchronization, and perhaps more advanced data visualization for long-term trends. The modular design makes it easy to add new features without breaking existing ones.

Conclusion

Building this personal automation dashboard has been an incredibly rewarding experience. It not only streamlines my daily digital interactions but also serves as a practical demonstration of how Python and Streamlit can be combined with various APIs to create powerful and personalized tools. I encourage you to explore the GitHub repository, experiment with the code, and perhaps even build your own!
